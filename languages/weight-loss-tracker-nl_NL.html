<strong>Measurements.</strong> Support for recording measurements like Hip, Waist, Leg, Bicep, etc. Displayed on charts and tables.
<strong>BMI.</strong> Allows a user to specify their height. Once specified, their BMI is displayed next to each weight entry. There is also a shortcode to render the latest BMI.
<strong>Widgets.</strong> Widgets that allow you to display the graph and quick weight entry form within any widget area. <strong>Widgets.</strong> Met deze widgets zie je snel en makkelijk het gewicht in de widget area.
<strong>Chart and form Shortcodes.</strong> That allow you to display the graph and quick weight entry form by placing a shortcode on any post or page. <strong>Grafieken en vormen codes.</strong> Dit geeft je toestemming om de grafiek en het gewicht wat je hebt ingevult te plaatsen op elke post pagina.
<strong>Text Shortcodes.</strong> Additional shortcodes for earliest and most recent dates entered. <strong>Tekst codes.</strong> Extra shortcodes voor de meest recente data wat je hebt ingevult.
<strong>Admin: View / Delete user data</strong>. Admin will be able to view and delete existing user data. <strong>Admin: Bekijk / bewerk gebruikers data</strong>.Je kan als admin gebruikers data bekijken, veranderen en verwijderen
<strong>User preferences</strong>. If enabled, the user will be able to select which unit they wish to store their weight in Metric or Imperial. They will also be able to specify date format and clear all their weight data. <strong>Gebruikers voorkeueren</strong> wanneer geselecteerd, kies je voor het opslaan van je gewicht in Meteric of Imerial. Je kan bepaalde datums en bepaald gewicht verwijderen.
<strong>Bar Charts</strong>. Fancy something different to a line chart? The plugin will also support Bar Charts. <strong>Staafdiagram</strong>. Iets anders dan een lijn diagram? Met deze plugin kan je kiezen voor een staafdiagram.
<strong>Decimals</strong>. Decimals will be allowed weight in Pounds only or Kg modes. <strong>Decimalen</strong>. Decimalen zijn alleen te zien in de Ponden of Kg modes.
<strong>Delete existing entry</strong>. A logged in user will be able to delete or edit an existing weight entry.
<strong>Better Tables.</strong>. Data tables in front end and admin will support paging and sorting. <strong>Betere tabellen.</strong>. De data in de tabellen in front & and admin zullen alles ondersteunen
<strong>Admin: Extra Settings</strong>. Extra settings to customise the plugin will be added e.g. number of plot points on graph, rows per page, etc. <strong>Admin: Extra Instellingen</strong> Extra instellingen om deze plugin aan te passen o.a. aantal nummers in de grafies, rijen per pagina, ect.
