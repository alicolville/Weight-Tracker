<strong>Admin: View / Edit user data</strong>. Admin will be able to view, edit and delete existing user data.
<strong>User can choose Unit</strong>. The user will be allowed to select which unit they wish to store their weight in Metric or Imperial.
<strong>Bar Charts</strong>. Fancy something different to a line chart? The plugin will also support Bar Charts.
<strong>Decimals</strong>. Decimals will be allowed weight in Pounds only or Kg modes.
<strong>Delete existing entry</strong>. A logged in user will be able to delete an existing weight entry.
<strong>Better Tables.</strong>. Data tables in front end and admin will support paging and sorting.
