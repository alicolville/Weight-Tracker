Disable user stats? Disabilitare le statistiche degli utenti?
User stats are maintained to help power shortcodes such as [weight-loss-tracker-league-table]. This involves a cron job being run every hour to refresh a user's stats such as first weight, recent weight and total lost. The figures are also re-produced on each weight entry modification and can then be used to generate a league table. This feature could <em>potentially</em> reduce performance on your site. If you are not using the leaderboard shortcodes it is advised to disable this. Statistiche per utente sono mantenuti per aiutare gli shortcode come [weight-loss-tracker-league-table]. Si tratta di un processo di cron che viene eseguito ogni ora per aggiornare le statistiche di un utente come primo peso, il peso recente e totale perdita. Le figure sono anche ri-prodotte su ogni modifica peso e possono quindi essere utilizzati per generare una classifica. Questa caratteristica potrebbe <em>potenzialmente</em> ridurre le prestazioni del tuo sito. Se non si utilizzano gli shortcode si consiglia di disabilitare questo.
Send usage data to YeKen? Invia i dati di utilizzo di YeKen?
We would love to see how our plugin is used. By consenting, you are allowing permission for the following data to be sent on a monthly basis to YeKen: URL, summary of settings, count of recorded weights and measurements. No user or admin data will ever be transmitted. Ci piacerebbe vedere come viene usato il nostro plugin. Si consente il permesso per i seguenti dati da inviare su base mensile a YeKen: URL, riepilogo delle impostazioni, conte di pesi e misure registrate. verranno mai trasmessi dati utente o amministratore.

<strong>Measurements.</strong> Support for recording measurements like Hip, Waist, Leg, Bicep, etc. Displayed on charts and tables. <strong>Misure.</strong> Il supporto per le misurazioni di registrazione come Anca, Vita, Gamba, Bicipite, ecc Visualizzato su grafici e tabelle.
<strong>BMI.</strong> Allows a user to specify their height. Once specified, their BMI is displayed next to each weight entry. There is also a shortcode to render the latest BMI. <strong>BMI.</strong> Consente all'utente di specificare la loro altezza. Una volta specificato, il loro indice di massa corporea viene visualizzato accanto a ciascuna voce di peso. C'è anche un shortcode per rendere l'ultimo indice di massa corporea.
<strong>Overall user stats.</strong> Shortcodes that allow you to display the total lost / gained for the community and another to display a league table. <strong>Statistiche complessiva dell'utente.</strong> Shortcodes che consentono di visualizzare il totale perso / guadagnato per la comunità e un altro per visualizzare una classifica.
<strong>Widgets.</strong> Widgets that allow you to display the graph and quick weight entry form within any widget area. <strong>Widget.</strong> Widget che consentono di visualizzare il grafico e scheda di iscrizione di peso rapida all'interno di qualsiasi area del widget.
<strong>Chart and form Shortcodes.</strong> That allow you to display the graph and quick weight entry form by placing a shortcode on any post or page. <strong>Grafico e la forma Shortcodes.</strong> Che consentono di visualizzare il grafico e la scheda di iscrizione di peso rapida mettendo un codice funzione in qualsiasi post o pagina.
<strong>Text Shortcodes.</strong> Additional shortcodes for earliest and most recent dates entered. <strong>Testo Shortcodes.</strong> codici brevi aggiuntivi per le date primi e più recenti sono entrati.
<strong>Admin: View / Delete user data</strong>. Admin will be able to view and delete existing user data. <strong>Admin: Visualizza / Elimina dati utente.</strong> Admin sarà in grado di visualizzare e cancellare i dati utente esistenti.
<strong>User preferences</strong>. If enabled, the user will be able to select which unit they wish to store their weight in Metric or Imperial. They will also be able to specify date format and clear all their weight data. <strong>Preferenze dell'utente.</strong> Se abilitato, l'utente sarà in grado di selezionare quale unità desiderano conservare il loro peso in Metrico o Imperiale. Essi saranno inoltre in grado di specificare il formato della data e cancellare tutti i dati di peso.
<strong>Bar Charts</strong>. Fancy something different to a line chart? The plugin will also support Bar Charts. <strong>Grafici a barre.</strong> qualcosa di fantasia diverso da un grafico a linee? Il plugin sosterrà anche grafici a barre.
<strong>Decimals</strong>. Decimals will be allowed weight in Pounds only or Kg modes. <strong>Decimali.</strong> I decimali saranno ammessi peso in modalità Kg soli o Sterline.
<strong>Delete existing entry</strong>. A logged in user will be able to delete or edit an existing weight entry. <strong>Elimina voce esistente.</strong> Un utente connesso sarà in grado di eliminare o modificare una voce di peso esistente.
